---

# Available Endpoints

| Method | Endpoint      | Description      |
| ------ | ------------- | ---------------- |
| GET    | /workers      | Get all workers  |
| GET    | /workers/{id} | Get worker by id |
| POST   | /workers      | Create worker    |
| PUT    | /workers/{id} | Update worker    |
| DELETE | /workers/{id} | Delete worker    |
